# AIChatExport

一个 Edge/Chrome 浏览器扩展，用于选择和导出 AI 对话历史记录为 Markdown 文件。

## 功能

- **一键选择模式** — 点击标题栏的 📥 按钮，进入对话选择模式
- **按需勾选** — 每条对话前出现复选框，自由选择要导出的内容
- **全选/取消** — 底部操作栏支持全选和取消
- **自动跟踪新消息** — 选择模式下，新加载或新增的对话自动带上复选框
- **状态持久化** — 勾选状态存储在浏览器本地，刷新页面不丢失（15 分钟自动过期）
- **导出为 .md** — 将选中的对话导出为规范的 Markdown 文件
- **Markdown 格式保留** — 标题、粗体、代码块、表格、数学公式等格式完整保留
- **角色自动识别** — 自动区分用户消息和 AI 回复

## 安装

### 从源码安装（开发者模式）

1. 下载或克隆本仓库到本地
   ```bash
   git clone https://github.com/UESteveJobs/AIChatExport.git
   ```
2. 打开浏览器扩展管理页面
   - Edge: 地址栏输入 `edge://extensions/`
   - Chrome: 地址栏输入 `chrome://extensions/`
3. 打开右上角的 **开发者模式**
4. 点击 **加载已解压的扩展**，选择项目文件夹
5. 插件图标出现在工具栏中，安装完成

### 更新

在扩展管理页面找到插件卡片，点击 **重新加载** 即可更新。

## 使用

1. 打开 DeepSeek 聊天页面（`https://chat.deepseek.com/`）
2. 进入一条有历史对话的会话
3. 点击标题栏右侧的 **📥 按钮**
4. 页面进入选择模式：
   - 每条用户消息前出现复选框
   - 底部输入框隐藏，替换为操作栏
5. 勾选你想要导出的对话（支持全选）
6. 点击 **导出为 .md**
7. 浏览器自动下载 Markdown 文件
8. 选择模式自动退出，页面恢复原样

> **提示**：在选择模式下上划可以加载更多历史对话，新出现的对话会自动带上复选框。

## 技术细节

### 唯一标识

每条对话通过 DOM 属性 `data-virtual-list-item-key` 作为唯一标识，不依赖 DOM 位置，因此不受虚拟滚动影响。

### 状态管理

- 选中状态存储在 `chrome.storage.local`，刷新页面可恢复
- 缓存 15 分钟自动过期，防止残留数据堆积
- 页面关闭时自动清理缓存

### 内容提取

AI 回复内容通过递归方式遍历 DOM 树，以下格式会正确转换为 Markdown：

| HTML 元素        | Markdown 输出         |
| -------------- | ------------------- |
| `<h1>~<h6>`    | `#` \~ `######` 标题  |
| `<p>`          | 段落                  |
| `<strong>/<b>` | `**粗体**`            |
| `<em>/<i>`     | `*斜体*`              |
| `<code>`       | `` `行内代码` ``        |
| `<pre>`        | ` ```代码块``` `       |
| `<blockquote>` | `> 引用`（支持嵌套）        |
| `<ul>/<ol>`    | 无序/有序列表（支持嵌套）       |
| `<table>`      | Markdown 表格         |
| `<del>/<s>`    | `~~删除线~~`           |
| KaTeX 公式       | `$...$` 或 `$$...$$` |
| SVG/MATH       | 自动忽略                |
